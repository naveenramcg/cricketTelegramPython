# cricketTelegramPython
A python telegram bot that fetches the cricket score of live matches.
Python is an open source language that was made to be easy to read and powerful.
This project uses messaging app- Telegram to create a bot, using the api of bot to automate it.
The api is used to access the messages being send to bot and reply to the messages automatically.
The bot we have created access the cricket scores by pypi package called pycricbuzz.
